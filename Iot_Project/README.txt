1. Подключить контроллер к питанию
2. Спустя примерно 30 секунд появится сеть "temperature_esp"
3. Из настроек подключения взять адрес DNS-сервера IPv4 (должен быть таким 192.168.4.1)
4. Ввести это адрес в браузере.
5. На открывшейся странице ввести логин и пароль WIFI, ip и порт MQTT сервера 
6. Сохранить введённые данные и перезагрузить устройство.
7. При корректно введённых данных ESP8266 начнёт отправлять данные в топике EC:FA:BC:75:C0:1C (его mac адрес)

-----------------------------

Для того чтобы поменять файлы в файлововой системе (папка data) нужно открыть скетч
с помощью старой версии Arduino, выбрать пункт инструменты, выбрать пункт
ESP8266 Sketch Data Upload. При этом монитор порта должен быть закрыт

Сообщение о обновлении библиотек нужно проигнорировать, поскольку работа с json файлом
реализована на старой версии библиотеки

Вся инфориация при работе микроконтроллера выводится в монитор порта
